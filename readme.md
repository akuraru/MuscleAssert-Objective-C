# MuscleAssert

A unit test framework for Objective-C using computation expressions.

## Proposal

What is `MuscleAssert`

* provide an assertion message that makes the difference more obvious.
* is not "where is it going?" But "What's different?"

## Usage

```
NSArray *left = @{
    @"user": @{
        @"name" : @"akuraru",
    }
};
NSArray *right = @{
    @"user": @{
        @"name" : @"",
    }
};
```

Result

```
path: .user.name.0
left: akuraru
right: 
```
